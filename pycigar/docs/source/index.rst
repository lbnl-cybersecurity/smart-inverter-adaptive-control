Welcome to PyCIGAR
=================================

`PyCIGAR <https://pycigar-project.github.io>`_ is a computational framework for deep RL and control experiments for electrical grid.

PyCIGAR is a work in progress - input is welcome. Available documentation is limited for now.
`Tutorials <https://github.com/toanngosy/pycigar/tree/master/tutorials/exercises>`_ are being built in iPython notebook format.
